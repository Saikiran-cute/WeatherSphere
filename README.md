# 🌦️ WeatherSphere

A modern weather forecasting web application built using the MERN stack.

## Features

- 🌍 Search weather by city
- 📍 Current location weather
- 🌡️ Real-time temperature
- 💧 Humidity
- 🌬️ Wind speed
- 👀 Visibility
- 🌅 Sunrise & Sunset
- 📊 Air Quality Index (AQI)
- ⏰ Hourly Forecast
- 📅 5-Day Forecast
- 🎨 Dynamic weather-based UI
- 📱 Responsive Design

## Tech Stack

### Frontend
- React
- Vite
- Axios
- React Icons
- Framer Motion

### Backend
- Node.js
- Express.js
- OpenWeather API

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
